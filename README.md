# Cookie Alert

Cookie Alert for LOPD

## Descripción

Muestra una alerta para mostrar las cookies y si el usuario acepta o hace scroll la alerta desaparece y se guarda en memoria para que no vuelva a aparecer

## How to install

````
npm install cookiesAlert
````
